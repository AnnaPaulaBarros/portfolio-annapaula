Portfólio | Anna Paula Barros

Bem-vindo(a) ao meu portfólio!
Este projeto reúne meus principais trabalhos em Desenvolvimento Web, Análise de Dados e Visualização, apresentados em um layout interativo inspirado no universo Mario Bros.

O objetivo é mostrar minhas habilidades técnicas de forma criativa, acessível e divertida, sem perder a clareza e a organização profissional.

## Sobre o Projeto

O portfólio funciona como um “mapa de fases”, onde cada projeto representa uma conquista diferente:

Sites institucionais

Dashboards interativos

Projetos de Data Science

Visualizações de dados

Experimentos com Machine Learning (LDA / Topic Modeling)

Tudo com design responsivo, animações leves e interação dinâmica em JavaScript.

 ## Projetos em Destaque

 Idalina Barros | Paisagismo
Site institucional com formulário, depoimentos dinâmicos e integração com Firebase.

 Dashboard Interativo
Visualização de dados com foco em clareza, métricas e usabilidade.

Topic Modeling com LDA
Análise textual utilizando Latent Dirichlet Allocation para descoberta de temas.

 Projeto Adoção
Interface voltada para impacto social e experiência do usuário.

 Libertadores – Análise de Dados
Exploração e visualização de dados esportivos.

## Tecnologias Utilizadas

HTML5 – Estrutura semântica

CSS3 – Layout responsivo e tema retrô

JavaScript (ES6) – Interatividade e lógica

Firebase Firestore – Armazenamento de depoimentos

SweetAlert2 – Feedback visual ao usuário

Google Fonts – Press Start 2P, Poppins e Playfair Display
